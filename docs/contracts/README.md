# Contracts

Behavioral contracts: NFRs, SLOs, API contracts, error semantics, monitoring contracts.

Specs and ADRs reference contracts here. Forgecraft gates (e.g. `nfr-contracts-required`, `monitoring-contracts-missing`) probe this directory.

Legacy contracts live at `docs/TechSpec.md` and `docs/nfr-contracts.md` — see `docs/manifest.yaml`.
